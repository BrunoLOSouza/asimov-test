## 📂 Estrutura do Projeto

O repositório está dividido em duas frentes independentes:

### 🎨 Parte 1: Landing Page Positivus
**Objetivo:** Reprodução "pixel-perfect" das 4 primeiras seções da landing page "Positivus".
- **Fidelidade:** Uso rigoroso da paleta de cores branding e tipografia *Space Grotesk*.
- **Responsividade:** Layout adaptável para dispositivos móveis e desktops.
- **Interatividade:** Micro-interações em cards e botões.

### ⚡ Parte 2: Asimov Academy (Criação Autoral)
**Objetivo:** Desenvolvimento de uma Hero Section de alto nível para o curso de Python + IA.
- **Conceito:** Estética moderna inspirada em designs *premium* (Linear/Glassmorphism).
- **Recursos:** Efeitos de iluminação progressiva, gradientes suaves e transparências.
- **Copy:** Focado em conversão e clareza de proposta de valor.

## 🤖 Uso de Inteligência Artificial

Conforme solicitado, seguem as respostas sobre o processo de desenvolvimento assistido:

**1. Quais ferramentas de IA você usou?**
Utilizei o **Antigravity (AI Coding Assistant)** para a geração de código base, estruturação do projeto e auxílio na análise técnica. Para as artes visuais, usei modelos de geração de imagem integrados para criar ilustrações personalizadas.

**2. Em que etapas a IA ajudou mais?**
A IA foi fundamental na análise do Figma (mapeamento de cores e tipografia), na configuração rápida do ambiente com **Tailwind CSS v4** e na automação de tarefas repetitivas. Na Parte 2, ajudou a conceber o layout moderno de alto padrão.

**3. O que você ajustou manualmente e por quê?**
Ajustei manualmente as configurações de build do PostCSS, refinei espaçamentos para garantir o "pixel-perfect" e personalizei classes de gradiente e sombra para um aspecto mais fluido e premium, impossível de ser replicado puramente por automação.

## ⚙️ Como executar localmente

Se desejar rodar o projeto em sua máquina:

```bash
# Clone o repositório
git clone https://github.com/BrunoLOSouza/asimov-test.git

# Acesse uma das partes
cd parte-1 # ou cd parte-2

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```
