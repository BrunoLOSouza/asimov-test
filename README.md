# Teste Técnico - Designer / Frontend Dev (Asimov Academy)

Este repositório contém a resolução do teste técnico, dividido em duas partes principais conforme solicitado.

## Respostas sobre o uso de IA

**1. Quais ferramentas de IA você usou?**
Utilizei o **Antigravity (AI Coding Assistant)** para a geração de código, estruturação do projeto e auxílio na análise técnica. Para as artes visuais, usei modelos de geração de imagem integrados para criar ilustrações personalizadas que respeitassem a identidade visual proposta.

**2. Em que etapas a IA ajudou mais?**
A IA foi fundamental na análise do Figma (mapeamento de cores e tipografia), na configuração rápida do ambiente com Vite + Tailwind CSS v4 e na geração de ilustrações "line art" exclusivas para a Parte 1. Na Parte 2, a IA acelerou a criação de um layout moderno inspirado em referências de alto padrão como Linear.

**3. O que você ajustou manualmente e por quê?**
Ajustei manualmente as configurações de build do PostCSS para garantir compatibilidade com o Tailwind v4 e refinei os espaçamentos (paddings/margins) para garantir o "pixel-perfect" em diferentes resoluções. Também personalizei as classes de gradiente e sombra para que a interface tivesse um aspecto mais premium e fluido.
